nvm

node版本管理 开始前需卸载node

下载

nvm最新下载地址
选择 nvm-setup.zip 下载

下载成功后双击安装，自主选择选择nvm安装路径、node安装路径，开始安装。
打开CMD，输入命令 nvm ，安装成功则如下显示。可以看到里面列出了各种命令

命令

1、查看本地安装的所有版本；有可选参数available，显示所有可下载的版本。


2、安装，命令中的版本号可自定义，具体参考命令1查询出来的列表

3、使用特定版本

4、卸载

nrm

nrm(npm registry manager ) node镜像源管理工具

下载

全局安装 nrm

命令

1.查看可选源


2.切换源
切换到taobao源

3.增加源
你可以增加定制的源，特别适用于添加企业内部的私有源，执行命令 nrm add ，其中reigstry为源名，url为源的路径。

4.删除源
执行命令nrm del 删除对应的源

4.测试速度
通过 nrm test 测试相应源的响应时间
